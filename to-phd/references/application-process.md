# 国内博士申请流程管理

## 先确认具体制度

不同学校、学院和专业可能采用申请考核制、普通招考、硕博连读、直博或专项计划。先从当年官方文件确认申请类型，再建立流程，不把一种制度套用到所有项目。

## 阶段划分

### 1. 申请准备

建立申请者档案、目标清单、主简历、成果证明、成绩与学位材料、推荐人安排和时间表。

### 2. 信息核验与导师联系

核验当年简章、学院细则、专业目录、导师方向和招生状态；记录联系、回复与待确认事项。

### 3. 网上报名与材料提交

逐项记录报名系统、开放与截止时间、缴费、纸质或电子材料、签字盖章、公证或认证要求，以及材料是否被确认接收。

### 4. 资格审核与考核

记录审核状态、补件要求、笔试、面试、科研计划陈述、英语考核、思想政治考核和其他学院特定环节。

### 5. 结果与后续

记录成绩、拟录取、公示、调档、政审、体检、定向或非定向手续、录取通知及入学前任务。

## 任务追踪规则

每项任务至少包含：

`目标｜阶段｜任务｜官方截止时间｜内部截止时间｜负责人｜状态｜所需材料｜证据链接｜最后核验日期｜下一步`

推荐状态：`未开始`、`进行中`、`等待外部回复`、`待核验`、`已完成`、`已放弃`。

## 材料检查

- 以当年官方清单为准，不凭记忆复用上一所学校的清单。
- 检查文件格式、命名、页数、签字、盖章、上传位置和提交方式。
- 识别同一材料在学校层面与学院层面的不同要求。
- 在官方截止时间前设置内部截止时间，给补件和系统故障留出余量。
- 对尚未发布的环节明确标记“等待当年通知”，不要使用往年日期代替。

## 每次更新后的输出

1. 当前处于哪个阶段。
2. 最近的硬截止时间。
3. 本周最重要的三项行动。
4. 等待他人或机构回复的事项。
5. 可能导致失去资格的风险。
